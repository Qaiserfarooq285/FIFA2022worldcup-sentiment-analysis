## **Description**  
This repository contains a **sentiment analysis project** based on a dataset of **22,000 tweets** collected during the FIFA World Cup 2022. The analysis provides insights into public opinion about the tournament, classifying the percentage of **positive** and **negative** reviews.

---

## **Features**  
- Dataset of 22,000 tweets from the FIFA World Cup 2022.  
- Sentiment classification to determine public opinion.  
- Percentage breakdown of positive and negative sentiments.
